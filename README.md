# Parcel introduction

Settings to use the following in the parcel.

- Sass
- Autoprefixer
- Babel
- Local server with Hot Module Replacement

## Compiles and hot-reloads for development

```
npm run serve
```

## Compiles and minifies for production

```
npm run build
```
